Atajos de teclado en Word y Google Docs

¡Trabajá más rápido y como un pro!

📝 Básicos que tenés que saber:
🔹 Ctrl + C / Ctrl + V → Copiar y pegar
🔹 Ctrl + Z → Deshacer un error
🔹 Ctrl + B → Texto en negrita o Buscar
🔹 Ctrl + G → Guardar (¡fundamental!)
🔹 Ctrl + P → Imprimir

💡 En Google Docs, además podés usar:
🔹 Ctrl + Alt + M → Agregar comentario
🔹 Ctrl + K → Insertar enlace

📌 Y hay muchos mas - todo depende tu region de teclado

Google Drive vs OneDrive… ¿cuál te conviene más?

📌 Google Drive
- Integración total con Gmail y Google Docs
- Ideal para trabajar en equipo en tiempo real
- 15 GB gratis de inicio

📌 OneDrive
- Perfecto si usás Windows y Microsoft Office
- Sincronización directa con tu PC
- Incluido con suscripción a Microsoft 365
